~~~ These are done on the virtual machine ~~~~

Generate crypto files:
******************************************************************************************************************************************************
cryptogen generate --config=./crypto-config.yaml
******************************************************************************************************************************************************

Create the channelTx file
******************************************************************************************************************************************************
configtxgen -profile TwoOrgsChannel -outputCreateChannelTx ./channel-artifacts/ottospis.tx -channelID ottospis
******************************************************************************************************************************************************

Create the genesis block
******************************************************************************************************************************************************
configtxgen -profile SampleMultiNodeEtcdRaft -channelID ottospis -outputBlock ./channel-artifacts/genesis.block
******************************************************************************************************************************************************
